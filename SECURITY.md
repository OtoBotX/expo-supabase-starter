# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| develop | ✅ Supported        |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by [creating a new GitHub issue](../../issues/new/choose) using the appropriate template.

When reporting, please include as much detail as possible, including:

- A clear description of the vulnerability
- Steps to reproduce the issue
- Potential impact or risk

We aim to review and respond to all security reports within **5 business days**. If the vulnerability is confirmed, we will work on a fix promptly and keep you updated through the issue thread.

Thank you for helping make this project more secure!
